## OpenAPI preview

We preview the OpenAPI contract using the Swagger Viewer extension in VS Code.

1. Install the `Swagger Viewer` extension if you do not have it.
2. Open [docs/openapi/openapi.yaml](docs/openapi/openapi.yaml).
3. Run **Swagger Viewer: Preview Swagger** from the Command Palette.

Notes:

- The preview is read-only; edit YAML files directly, then re-run the preview.
- If referenced files change (for example, components or paths), refresh the preview tab.
